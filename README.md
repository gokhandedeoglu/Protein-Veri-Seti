# Protein Veri Seti

  Protein veri seti üzerinde yaptığım analizleri içeren çalışma.
  Başlangıç olarak kaggle'dan alınan rastgele alınan veri seti üzerindde çalışmak üzere porjeye başladık.
  Proteinlerin yapısal özellikleri ile ilgili veri setini anlamaya, çözümlemeye, görselleştirmeye ve eksik verileri doldurma üzerine çalışma yaptık.
  
  ## İçerik
  
  ### 1) Verinin Keşfi (EDA)
  + #### 1.1) Kategorik Değişkenler
  + #### 1.2) Sürekli Değişkenler
   
  ### 2) Verinin Görselleştirilmesi (Data visualization)
  + #### 2.1) Missing Value Görselleştirme
  + #### 2.2) phValue Görselleştirmesi
   
  ### 3) Aykırı Değer Temizleme(Outlier Cleaning)
   + #### 3.1) Baskılama Yöntemi
   
  ### 4) Eksik Veri Doldurma
